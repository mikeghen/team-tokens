# Team Tokens Data Analysis

## Historical Price Analysis
- Review the hypothetical investment of 10K at the begining of the season using the Team Tokens fixed 2% betting strategy.
- Data Needs:
    - For each nba team in the 2025-26 season:
        - For each game for that team:
            - Hourly price history for the YES/NO tokens at least 48 hours leading up to the event
- Hypothetical investment plan:
    - Simple investments of 2% of a fund, split into 4 chunks done 48 hours, 36 hours, 24 hours, and 12 hours ahead of the game time.

## Data Sources
- Polymarket API
- Dome API
